2024 Final Year Project for CEG311 NUSricq

The **project scope** includes:

(1) Conducting a review of existing open-source educational simulation tools for computer architecture;

(2) Identifying existing tools that can be used as a starting point (e.g., RARS simulator);

(3) Incorporating structural (block-level design) visualisation into the simulator;

(4) Incorporating cycle-approximate performance information for high-performance designs (if time permits);

(5) Iterative improvement of the simulator from user feedback.
